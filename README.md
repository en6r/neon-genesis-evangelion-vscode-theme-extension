## Installation
1. Install and launch [Visual Studio Code](https://code.visualstudio.com/)
2. *Settings* > *Extensions* (or `Ctrl+Shift+X`)
3. Search for `Neon Genesis Evangelion Themes`
4. Click *Install*
5. *Settings* > *Themes* > *Color Theme* > `EVA Unit-01 Berserk`
